## ¿Qué es theWALL?

theWALL es una red social desarrollada en PHP con MySQL.
Aquí podrás publicar texto e imagenes, compartirla con tus contactos, personalizar tu perfil, vissitar el perfil de otra persona, entre otras cosas.
